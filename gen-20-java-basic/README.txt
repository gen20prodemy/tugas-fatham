Code :
public class HelloWord { //Nama Kelas
    public static void main (String[] args){ //Method Main
        System.out.println("Hello World"); //Print text and new line to console
        System.out.print("Hallo Dunia");//Print text to Console
    }
}

Penjelasan :
1. public class HelloWorld merupakan Nama kelas dari file .java yang telah dibuat sebelumnya, nama kelas dengan nama file .java harus sama agar dapat di eksekusi. Nama kelas selanjutnya diikuti dengan tanda kurung kurawal. Method atau code diletakkan didalam kelas, yaitu didalam kurung kurawal (tanda sebagai awal dan akhir kelas).
2. public static void main (String[] args) merupakan method main. Metod main merupakan method utama, ini berfungsi untuk menjalankan kelas. Jika dalam class tidak terdapat method main maka class tidak dapat dijalankan.
3. System.out.println : Command untuk print text dan new line.
System.out.println("Hello World"); berarti, console akan mencetak text : Hello World dan selanjutnya pointer akan diletakkan di new line
4. System.out.print : Command untuk print text saja
System.out.print("Hallo Dunia"); berarti console akan mencetak text : Hallo Dunia dan selanjutnya pointer tetep di letakkan di sebelah kanan huruf "a" pada kata "Dunia"
